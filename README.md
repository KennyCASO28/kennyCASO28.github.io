Kiosco UTP: Plataforma Web de Pedidos Anticipados para el Campus (Prototipo HTML)
Este proyecto es un prototipo web funcional desarrollado como parte del curso  Taller de Programación Web.
Simula el sistema de pedidos de la cafetería de la Universidad Tecnológica del Perú (UTP), permitiendo a los estudiantes optimizar su tiempo y evitar aglomeraciones.
Características Destacadas:
Menú Dinámico: Visualización clara de productos categorizados (Especiales, Desayunos, Postres, etc.).

Carrito de Compras (Mini-Carrito): Permite añadir y gestionar productos de forma ágil desde el menú.

Checkout Completo: Flujo de compra simulado con desglose de totales (Base Imponible e IGV).

Diseño Responsivo (Mobile First): La interfaz está optimizada para su uso en dispositivos móviles y tabletas, facilitando la experiencia del usuario en el campus.

Estructura Semántica y Accesibilidad (WAI-ARIA): Uso de etiquetas semánticas y atributos ARIA para mejorar la navegación y accesibilidad.

desarrollado en: 
HTML5: Estructura de las páginas.

CSS3: Estilos responsivos y diseño moderno.

JavaScript (Vanilla JS): Lógica de negocio, gestión del carrito, carrusel de banners y validaciones del formulario.
